Permission

Super Admin

Super admin is the most powerfull role in the system. Has access to all feature in the Admin console, and manage every aspect of your organization's account

  • Create and assign permissions or requests
  • Manage other super and delegated admins such as profile , template, document ...
  • Can change password
  • Allow to access to all features of system
  • Can Read, Create, Update and Delete all features of system
  • Change system settings such as fonts or theme ...

Readonly Admin

You have access to all elements of the system. However, you are not allowed to change anythings including add new, update or delete. This is the right to help you have the most overview of the entire system.

  • Create and assign permissions or requests
  • Manage your personal profile (not admin profile)
  • Can change owner's password
  • Allow to access to all features of system
  • Can Create, Update and Delete all features of system
  • Change system settings such as fonts or theme ...

Human Resources

A company or organization's HR department is usually responsible for creating, putting into effect and overseeing policies governing workers and the relationship of the organization with its employees. HR can access all right relate to main features and activities of system

  • Create and assign permissions or requests
  • Manage other super and delegated admins such as profile , template, document ...
  • Can change owner's password
  • Allow to access to all features and activities of system
  • Can Read, Create, Update and Delete all features of system
  • Change system settings such as fonts or theme ...

Interviewers

You are a guest to attend one or more specific activities such as interview, call or video call. Other than that you can't do anything else on the system

  • Create and assign permissions or requests
  • Manage other super and delegated admins such as profile , template, document ...
  • Can change password
  • Allow to access to all features of system
  • Can access to jobs requisition, and something just related to this user
  • Change system settings such as fonts or theme ...
  • Join activities
